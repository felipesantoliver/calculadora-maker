# calculadora-maker

Calculadora de orçamentos para projetos de impressão 3D, desenvolvida para estimar custos de forma prática e precisa. A ferramenta considera variáveis essenciais como consumo de material, tempo de máquina, gasto energético e mão de obra.

Ideal para makers, engenheiros e prestadores de serviço que desejam padronizar e agilizar a precificação dos seus projetos.

📌 Sobre o projeto

Este projeto faz parte de uma série de estudos focados em Vibe Coding que estou realizando, explorando desenvolvimento prático, iteração rápida e construção orientada à experiência.
